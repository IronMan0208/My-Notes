# Notes App

A modern Android Notes App built using Kotlin and Jetpack Compose. This application allows users to create, edit, update, and delete notes with a clean and user-friendly interface.

## Features

* Create Notes
* Edit Notes
* Delete Notes
* View All Notes
* Clean and Modern UI
* Responsive Design

## Tech Stack

* Kotlin
* Jetpack Compose
* Material 3
